# 와이어프레임 (텍스트 기반 대시보드)

이 와이어프레임은 시스템 관리자(예: 김사장 페르소나)가 웹이나 앱을 통해 현재 시스템 상태를 모니터링하는 화면을 텍스트로 표현한 것입니다.

```plaintext
/----------------------------------------------------------------------
|         엣지 VLM 지능형 공조 제어 시스템 - 대시보드 (에너자이즈 피트니스)         |
|----------------------------------------------------------------------|
|                                                                      |
| [실시간 공간 분석 (VLM)]                  [현재 공조 상태 (IoT)]        |
|   - 현재 인원: 8명                        - 실내 온도: 24.5°C        |
|   - 활동 수준: 높음 (High Met)            - 실내 습도: 55%           |
|   - 평균 착의량: 가벼움 (Light Clo)       - 동작 모드: 냉방 ❄️       |
|   - 예측 열 쾌적도(PMV): +1.2 (Slightly Warm) - 풍량: 강풍 (3/3)          |
|                                                                      |
|----------------------------------------------------------------------|
|                                                                      |
| [에너지 절감 현황]                                                     |
|   - 오늘 절감된 예상 전력량: 15 kWh                                    |
|   - 이번 달 누적 절감액: ₩ 45,000                                      |
|   - 실시간 전력 소모량: 2.1 kW                                         |
|                                                                      |
|----------------------------------------------------------------------|
|                                                                      |
| [최근 시스템 이벤트 로그]                                                |
|   - [14:32] VLM: 고강도 활동 감지. 냉방 모드 및 풍량 자동 상향 조정.  |
|   - [14:15] VLM: 인원 3명 감소 감지. 공조 효율 모드로 전환.            |
|   - [13:58] VLM: 창가 근처 사용자 더워하는 행동 감지. 해당 구역 풍향 집중. |
|   - [13:30] 시스템: 외부 기온 하강에 따라 목표 온도 자동 재설정.        |
|                                                                      |
\----------------------------------------------------------------------/
```

### 와이어프레임 설명
1.  **실시간 공간 분석 (VLM):**
    *   VLM이 엣지 디바이스에서 분석한 비식별 텍스트 데이터를 보여줍니다.
    *   관리자는 이를 통해 현재 공간의 상황을 직관적으로 파악할 수 있습니다.

2.  **현재 공조 상태 (IoT):**
    *   HVAC 기기에 연결된 IoT 센서가 측정한 정량적 데이터를 보여줍니다.
    *   VLM의 판단과 현재 기기 상태를 함께 비교하며 볼 수 있습니다.

3.  **에너지 절감 현황:**
    *   관리자의 가장 큰 관심사인 ROI(투자 대비 수익)를 시각적으로 보여주는 핵심 지표입니다.
    *   시스템 도입 후 얼마나 비용이 절감되고 있는지 실시간으로 확인할 수 있습니다.

4.  **최근 시스템 이벤트 로그:**
    *   VLM과 IoT 데이터 융합을 통해 시스템이 어떤 판단을 내리고 자동으로 제어했는지 기록을 보여줍니다.
    *   이를 통해 관리자는 시스템이 의도대로 잘 동작하고 있는지 신뢰할 수 있습니다.
