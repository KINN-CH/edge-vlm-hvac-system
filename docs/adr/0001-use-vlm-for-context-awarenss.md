# ADR-0001: VLM 기반 공간 맥락 인지 시스템 도입

## Status

Accepted

---

## Context

기존 HVAC 제어 시스템은 온도 및 습도 센서와 같은 정량 데이터에만 의존하여 동작한다.
이 방식은 다음과 같은 한계를 가진다:

* 재실자 수 변화 반영 어려움
* 활동량(Met), 착의량(Clo) 등 쾌적성 핵심 요소 측정 불가
* 입실/퇴실 등 상황(Context) 기반 제어 불가능

또한, 본 프로젝트는 **완전 오프라인 환경**에서 동작해야 하므로
클라우드 기반 AI 사용이 제한되는 제약이 존재한다.

이러한 문제를 해결하기 위해
**비정형 데이터(이미지)를 활용한 상황 인지 방식**이 필요하다.

---

## Decision

카메라 입력을 기반으로 VLM(Qwen2.5-VL)을 활용하여
공간의 맥락(Context)을 인지하는 시스템을 도입한다.

VLM을 통해 다음 정보를 추출한다:

* 재실자 수 (People Count)
* 활동량 (Met)
* 착의량 (Clo)
* 행동 패턴 (예: 퇴실 준비)

추출된 데이터는 IoT 센서 데이터와 결합하여
PMV 기반 HVAC 제어에 활용한다.

---

## Consequences

### 👍 장점

* 단순 센서 기반 대비 높은 표현력 확보
* 비정형 데이터(행동, 상황) 처리 가능
* 사용자 쾌적성(PMV) 예측 정확도 향상
* 오프라인 환경에서도 지능형 제어 가능

### 👎 단점

* 연산 비용 증가 (Jetson 리소스 부담)
* 모델 최적화 필요 (INT4, TensorRT)
* VLM 추론 정확도에 의존

---

## 📌 Purpose (목적)

이 ADR 문서는 다음을 목적으로 한다:

* **지식 전수:** VLM 도입 배경과 이유를 명확히 공유
* **반복 논쟁 방지:** 동일한 기술 선택에 대한 재논의 방지
* **온보딩 가속화:** 신규 팀원이 빠르게 시스템 이해 가능
